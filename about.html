Acerca de la Aplicación

Esta aplicación corresponde a la cuarta entrega de una App Web de Clima Internacional (MVP), correspondiente al Modulo V, desarrollada como parte de un portafolio académico.

**¡IMPORTANTE**: A diferencia de entregas anteriores con datos ficticios, esta versión utiliza **datos climáticos reales** obtenidos mediante la **API de OpenWeatherMap**, mostrando temperaturas actuales, pronósticos semanales y estadísticas precisas.

La idea que inspira este proyecto surge de la necesidad de ofrecer una **herramienta simple, visual e interactiva** para consultar el clima de 10 capitales internacionales, seleccionadas por su alto índice de felicidad mundial y ser destinos prioritarios de estudios de postgrado para estudiantes chilenos.

Estructura de la Aplicación

Home

Tarjetas flip-card interactivas con 10 capitales (paginación 5+5). Cada tarjeta muestra clima actual (temperatura, descripción, humedad, viento) y pronóstico de 4 días. Además de Modal con estadísticas completas.

Características Técnicas Implementadas

POO
Clase ApiClient + WeatherApp, Dependency Injection, Encapsulamiento
ES6+
Arrow functions, async/await, template literals, optional chaining
Asincronía
20 llamadas API paralelas (Promise.all), fetch nativo, try/catch
OpenWeatherMap
Clima actual + Forecast 5 días, datos en español, fallback inteligente
UX/UI
Flip cards 3D, paginación, modal responsive, loading states
Responsive
Bootstrap 5 + CSS Grid/Flexbox, funciona en cualquier dispositivo
